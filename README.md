A library for requesting and transforming Google Docs.

## Installation

Add this line to your application's Gemfile:

```ruby
gem "skoogle_docs"
```

And then execute:

```sh
$ bundle
```

Or install it yourself as:

```sh
$ gem install skoogle_docs
```
